# eCommerce

### Demo
[ecommerce_demo](https://ecommerce-live.herokuapp.com)

### Overview
This app can:
* login using google account
* view shop items
* add items to cart
* checkout with stripe api

### Dependecies
* Node.js
* React/Redux
* Redux Saga
* Firebase
* Stripe
* Styled-Components

### Installation
1. clone repo locally
1. cd into directory
1. run `npm install`
1. run `npm start`
1. enjoy
