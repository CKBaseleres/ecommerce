# eCommerce

### Demo
[DEMO](https://ecommerce-live.herokuapp.com)

### Overview
This app can:
* login using google account
* view shop items
* add items to cart
* checkout with stripe api

### Dependencies
* Node.js
* React/Redux
* Firebase

### Installation
1. clone repo locally
1. cd into directory
1. run `npm install`
1. run `npm start`
1. enjoy
